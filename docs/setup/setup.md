
# Quick Start

1. git clone ...
2. cp .env.example .env  → fill in OLLAMA_BASE_URL
3. docker run -p 8000:8000 chromadb/chroma
4. ollama pull llama3.2 && ollama pull nomic-embed-text
5. npm install && npm run dev
6. Allow notifications: System Settings/Notifications/Terminal-notifier/Alerts
7. Open VSCode — intervention fires after 30sec idle
