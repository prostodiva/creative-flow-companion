### Vision

# Creative Flow Companion

A telemetry-driven AI engineering mentor that learns from developer workflows and provides context-aware guidance through behavioral analysis, long-term memory, and AI-assisted coaching.

The system combines:
- activity and workflow tracking
- behavioral pattern analysis
- RAG-based memory retrieval
- local LLaMA integration
- contextual engineering coaching
- developer productivity intelligence

Creative Flow Companion is designed to function as an AI-native engineering companion that observes sessions, identifies patterns, retrieves relevant historical context, and delivers actionable recommendations to improve focus, learning, and software engineering decision-making.
